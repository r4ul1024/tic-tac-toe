# Tic-Tac-Toe

Simple terminal Tic-Tac-Toe game in C++ for two players.

## Structure
- src/ — source files  
- include/ — headers

## Compile
g++ -Iinclude src/main.cpp src/game.cpp -o tic-tac-toe

## Run
./tic-tac-toe

## Controls
- Menu: play, edit names, stats, exit  
- Moves: numbers 1-9
